# Advent of Code

Advent of code에 오신걸 환영합니다! 올 한해도 모두 고생 많으셨어요. 아직 한달이나 남은 2022년, advent of code로 올 한해 마무리를 해보는건 어떨까요?

# Instructions

1. Advent of Code는 adventofcode.com에서 12월 1일에 나오는 25개의 재밌는 퀴즈들을 풀면서 한 해를 마무리하는 최고의 방법중 하나에요.
2. 아래 마련된 github repo에 각자 좋아하는 언어로 문제풀이를 올리며 함께 해주세요.
3. List of Elves에 본인의 이름과 github id를 작성해주시면 깃헙페이지에 초대해드려요.
4. 본인의 이름이 포함된 브랜치로 해주시고 PR을 올려서 마스터로 merge 시켜주세요 (마스터는 직접 올리지 말아주세요).
5. root에서 각자의 이름이 들어간 폴더를 만들어 작업해주세요.
   1. 서로가 어떻게 하고 있는지 보기 쉽도록 리포 한개에서 하도록 해요.
   2. 서로의 코드는 변경하지 않기로 해요.
6. 우리 함께 하도록 해요. 초기세팅부터 시작해 문제풀이까지 어려움이 있으면 옆 동료에게 꼭 물어보시고 함께 문제를 해결하도록 해요.
